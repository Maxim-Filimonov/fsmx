# Fsmx

A Finite-state machine implementation in Elixir, with opt-in Ecto friendliness

## Installation

Add dictator to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:fsmx, "~> 0.1.0"}
  ]
end
```
